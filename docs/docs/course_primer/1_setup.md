# Setup

Get started by [installing the tools](https://use.ink/getting-started/setup).

## Create an ink! project

Create a template contract,

```shell
cargo contract new flipper
```

Test it,

```shell
cargo test
```

Comipile it,

```shell
cargo contract build --release
```
